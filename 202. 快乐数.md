## 解析

### 方法 1（快慢指针）判断是否有环

![](https://output66.oss-cn-beijing.aliyuncs.com/img/20220219204217.png)
使用 “快慢指针” 思想，找出循环：“快指针” 每次走两步，“慢指针” 每次走一步，当二者相等时，即为一个循环周期。此时，判断是不是因为 1 引起的循环，是的话就是快乐数，否则不是快乐数。

```js
/**
 * @param {number} n
 * @return {boolean}
 */
var isHappy = function (n) {
  let pre = n,
    cur = getNext(n);
  while (cur != pre && cur != 1) {
    pre = getNext(pre);
    cur = getNext(getNext(cur));
  }
  return cur === 1;
};
var getNext = (n) => {
  let t = 0;
  while (n) {
    t += Math.pow(n % 10, 2);
    n = Math.floor(n / 10);
  }
  return t;
};
```
